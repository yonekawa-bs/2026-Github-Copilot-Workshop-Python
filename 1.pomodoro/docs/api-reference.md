# API リファレンス

> **注意:** API は現時点では未実装です（STEP 3 で実装予定）。このドキュメントは実装後に更新されます。

## 実装予定のエンドポイント

Flask アプリ（`src/routes.py`）が実装されると、以下のエンドポイントが利用可能になる予定です。

| エンドポイント | メソッド | 役割 |
|---|---|---|
| `/` | GET | `index.html` を返す |
| `/api/stats` | GET | 今日の進捗（完了数・集中時間）を取得 |
| `/api/stats` | POST | ポモドーロ完了時に記録を追加 |
