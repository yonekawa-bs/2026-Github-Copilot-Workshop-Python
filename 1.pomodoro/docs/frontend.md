# フロントエンドドキュメント

> **注意**: 現時点ではフロントエンドは未実装です。`static/` ディレクトリおよび `templates/` ディレクトリは存在しません。

## 実装予定コンポーネント

設計ドキュメント（`architecture.md`）に基づく予定フロントエンド構成：

| 層 | ファイル | 責務 |
|---|---|---|
| 構造 | `templates/index.html` | セマンティックな HTML。円形プログレスバー用の SVG 要素を含む |
| 見た目 | `static/css/style.css` | 紫基調のグラデーション背景、カード型レイアウト、ボタンスタイル |
| 動作 | `static/js/timer.js` | カウントダウン制御、状態遷移、プログレスバー更新、API 呼び出し |

## 設計方針（予定）

タイマーのカウントダウンはフロントエンド（JavaScript）で処理する。サーバーとの通信遅延に左右されず正確にカウントダウンできるため。

### 予定 UI 要素

| UI 要素 | 実装方法 |
|---|---|
| 円形プログレスバー | SVG `<circle>` + CSS `stroke-dashoffset` アニメーション |
| 中央のタイマー表示 | `<span>` でオーバーレイ、JS で毎秒更新 |
| 状態ラベル | 「作業中」「休憩中」を状態に応じて切替 |
| 開始 / リセットボタン | `click` イベントで `timer.js` 関数を呼び出し |
| 今日の進捗セクション | `/api/stats` からデータ取得して表示 |

### 予定ボタン動作

- **「開始」ボタン**: タイマー開始 / 一時停止のトグル
- **「リセット」ボタン**: 現在のセッションを初期値に戻す
