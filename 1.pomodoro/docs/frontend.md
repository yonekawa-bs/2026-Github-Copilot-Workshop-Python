# フロントエンド ドキュメント

> **注意:** フロントエンドは現時点では未実装です（STEP 4・5 で実装予定）。このドキュメントは実装後に更新されます。

## 実装予定のファイル

| ファイル | 役割 |
|---|---|
| `templates/index.html` | メインページ（SVG 円形プログレスバー、タイマー表示、ボタン、進捗セクション） |
| `static/css/style.css` | UIスタイル（紫基調グラデーション背景、カード型レイアウト、ボタンデザイン） |
| `static/js/timer.js` | タイマー制御（カウントダウン、状態遷移、プログレスバー更新、API連携） |

## 実装予定の機能

### `timer.js`（STEP 5）

- `setInterval` による毎秒カウントダウン
- 開始 / 一時停止トグル
- リセット処理
- SVG `stroke-dashoffset` による円形プログレスバーのアニメーション
- セッション完了時に `POST /api/stats` で記録
- `GET /api/stats` で今日の進捗を取得・表示
